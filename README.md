# Flask-GraphQL demo

A simple web application that uses GraphQL to represent the data schema and
for querying the API.

## Usage

### Run directly

1. Create a Python virtualenv

2. `pip install -r requirements.txt`

3. `python app.py`

### or... run using docker

`docker-compose build`

`docker-compose up`

Once the application is running, navigate to the application at `http://localhost:8080`
